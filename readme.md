Midnight Firearms is a multi-framework firearms licensing system for FiveM that restricts weapon usage based on Discord roles.

The script verifies player permissions using the Discord API and allows administrators to grant or revoke firearm access through persistent database overrides.

Features include:

• Discord role-based firearm licensing
• Persistent MySQL override system
• Admin panel for managing permissions
• Multi-framework compatibility (QBCore, Qbox, ESX, Standalone)
• Inventory-agnostic enforcement
• Optional NUI management interface

Developed by MidnightMTB and distributed under Ottawa Region Scripts.
